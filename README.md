## ⚙️ Set up your Gemini API key

imaginAI needs your Gemini API key before it can run.

1. Find the `.env.local` file in the app folder
2. Open the file with Notepad
3. Find this line:
   `GEMINI_API_KEY=`
4. Add your Gemini API key after the equals sign
5. Save the file

Example:
`GEMINI_API_KEY=your_api_key_here`

If you do not have a Gemini API key yet, get one from your Google AI Studio account before you start the app.
